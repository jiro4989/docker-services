== アプリの起動

[source,bash]
----
docker-compose build --no-cache
docker-compose up
----

トップページを表示する。

[source,bash]
----
curl http://localhost:8000/
----

APIにリクエストする。

[source,bash]
----
curl http://localhost:8000/api/money
curl http://localhost:8000/api/insert/1234
curl http://localhost:8000/api/delete/1
----
